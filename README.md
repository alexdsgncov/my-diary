# 📔 My Diary — Личный дневник

Минималистичное офлайн-приложение личного дневника для Android. Данные хранятся исключительно на устройстве, интернет не используется.

## Функции

- ✦ Создание, редактирование и удаление записей
- ◆ 5 вариантов настроения для каждой записи  
- ♥ Добавление записей в избранное
- ◈ Теги для организации записей
- ◎ Полнотекстовый поиск по записям и тегам
- ● Группировка по месяцам
- ✸ Тёмная тема, минималистичный дизайн

## Быстрый старт

### 1. Установка зависимостей

```bash
cd my-diary
npm install
```

### 2. Запуск в Expo Go (для тестирования)

```bash
npx expo start
```

Отсканируйте QR-код через приложение **Expo Go** на телефоне.

## Сборка APK для Android

### Вариант А: Через EAS Build (рекомендуется)

1. Создайте аккаунт на [expo.dev](https://expo.dev)
2. Установите EAS CLI:
   ```bash
   npm install -g eas-cli
   eas login
   ```
3. Соберите APK:
   ```bash
   eas build --platform android --profile preview
   ```
4. Скачайте APK по ссылке из вывода команды.

### Вариант Б: Локальная сборка

1. Установите Android Studio и Android SDK
2. Создайте нативный проект:
   ```bash
   npx expo prebuild --platform android
   ```
3. Соберите APK:
   ```bash
   cd android
   ./gradlew assembleRelease
   ```
   APK будет в: `android/app/build/outputs/apk/release/app-release.apk`

## Данные и приватность

- Все данные хранятся **только на устройстве** через `@react-native-async-storage/async-storage`
- Приложение **не требует интернет-соединения**
- Данные не передаются никуда
- Удаление приложения удаляет все данные
